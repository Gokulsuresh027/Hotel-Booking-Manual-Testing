# 🧪 Hotel Booking Website - Manual Testing Project

## 📌 Project Overview

This repository showcases a comprehensive **Manual Testing** project conducted on a **Hotel Booking Website**. The project follows the **Software Testing Life Cycle (STLC)** and includes detailed test documentation prepared to ensure the quality, functionality, and usability of the application.
